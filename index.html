HaythServ — Cube 2: Sauerbraten server modification.

Installation

To install the product, you just have to type following code in your terminal:

git clone https://github.com/Gnaoprr/HaythServ.git ./HaythServ
cd HaythServ
chmod +x ./install.sh
./install.sh

Configuration

To configure the product, type following code in your terminal:

nano conf/server.conf

then edit the configuration file, set server name, message of the day and others, then save file.

Starting/Stopping the server

Starting the server

To start the server, just type the following code in your terminal:

./bin/server start

Stopping the server

To stop the server, just type

./bin/server stop

in your terminal.

Other things with the server

Update the server's configuration file

To update the configuration file, type

./bin/server updateconf

in the terminal.

Restart the server

And, also, to restart the server, just type

./bin/server restart

in the terminal.

Change the server's admin password

Just type

./bin/server password new_password

in your terminal to change the server's admin password.
